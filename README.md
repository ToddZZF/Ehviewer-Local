# Ehviewer-Local

（Developing! 开发中！）

Host the comic downloaded from Ehviewer locally

将从Ehviewer下载的漫画部署到本地

## Why/缘由

我一般在 Ehviewer 上看漫画，有喜欢的就下载下来，积少成多已有成千上万本，均存在固态硬盘中（手机空间不够），估摸着应该有 150G. 每次看都要点开文件夹才知道是哪部，而且也没有标签啥的，很难找到想看的。所以就计划写一个本地服务器，通过网页来浏览下载的漫画。

## Goals/目标

Goal-1

- 封面预览
- 查看漫画

Goal-2

- 标签
- 搜索
